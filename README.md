# ☁️ Cloudinary Image Transformer

MLH Global Hack Week GenAI 2025 — Cloudinary API Challenge

## Features
- 🎨 8 AI-powered image transformations (grayscale, sepia, blur, cartoonify, etc.)
- 👤 Smart face detection and cropping
- ✂️ AI background removal  
- 📐 Crop & resize with fill modes
- 🚀 Real-time transformation preview
- 🌐 Works with any Cloudinary cloud account

## Usage
1. Enter your Cloudinary cloud name
2. Enter an image public ID
3. Select a transformation
4. See the before/after comparison instantly!

## Tech Stack
HTML5 • CSS3 • Vanilla JavaScript • Cloudinary Transformation URLs API

Live demo: Open index.html in browser
